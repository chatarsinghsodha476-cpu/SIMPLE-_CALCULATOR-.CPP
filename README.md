# Simple Calculator in C++

## 📌 Overview

This project is a **Simple Calculator** developed in **C++**. It is a console-based application that allows users to perform basic arithmetic operations through a menu-driven interface. The program is beginner-friendly and demonstrates the use of conditional statements, loops, switch-case, and user input handling.

---

## 🚀 Features

- Addition of two numbers
- Subtraction of two numbers
- Multiplication of two numbers
- Division with divide-by-zero validation
- Find the maximum of two numbers
- Menu-driven interface
- Continue or exit option
- Invalid input handling

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project in any C++ IDE such as:
   - Dev-C++
   - Code::Blocks
   - Visual Studio Code
3. Compile the source code.
4. Run the program.
5. Enter two numbers.
6. Select an operation from the menu.
7. View the result.
8. Continue or exit as desired.
